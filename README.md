### A React WordPress theme rendered on the server with V8Js

![](http://joehoyle-captured.s3.amazonaws.com/2Fdon5iM.png)

#### Dependancies

- [PHP V8Js](https://github.com/phpv8/v8js)
- `yarn`

##### Install

```
cd wordpress-theme
yarn install
yarn run build

```

Activate the React WordPress theme.
